# 🚀 Job Helper Bot - ПАРСЕР VERSION

## ✅ РАБОТАЕТ! Парсит HH.ru

Теперь бот **парсит HTML страницы HH.ru** вместо API!

## 📋 Требования
- Python 3.10+

## 🔑 Токены
Вставь в `.env`:
- `BOT_TOKEN` - от @BotFather
- `OPENAI_API_KEY` - с platform.openai.com

## ⚡ Запуск

**Windows:** Двойной клик на `run.bat`
**Mac/Linux:** `bash run.sh`

## 🎯 Использование

1. `/start`
2. Заполни профиль
3. 🔍 Найти вакансии
4. Получи список!

**Готово! 🚀**
